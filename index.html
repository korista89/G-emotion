🔗 Google Sheets 연동 설정

Google Apps Script 웹앱 URL을 입력하세요:


📋 설정 단계:
1. Google Drive에서 새 스프레드시트 생성
2. Apps Script에서 제공된 코드로 웹앱 배포
3. 중요: 액세스 권한을 "모든 사용자"로 설정
4. 생성된 URL을 위에 입력 후 "Google Sheets 연결"
5. "연결 테스트"로 정상 작동 확인

💡 문제 해결:
• 연결 실패 시 F12 → Console 탭에서 오류 확인
• Apps Script에서 "새 배포" (기존 배포 수정 ❌)
• URL이 /exec로 끝나는지 확인
